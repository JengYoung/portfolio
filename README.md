## 🚀 **배경**

가끔은 글이 모든 것을 보여줄 수 없다고 생각해요.  
그렇기에 언젠가, 한 번 제 포트폴리오를 제작해보고 싶다는 생각을 하고는 했어요.

마침 이번에 다시 취직 준비를 시작하게 된 겸, 동기와 흥미가 있는 지금이 적기라 생각하여 포트폴리오 웹사이트를 만들어 보았어요.
